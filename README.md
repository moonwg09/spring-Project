# spring-Project

#### 스프링 프로젝트(감귤마켓)

# 💻 프로젝트 소개
당근마켓을 참고하여 만든 중고거래사이트입니다.

# 🕒 개발기간
* 23.10.30 ~ 23.12.07

# 👥 멤버구성
박용호- 전체 페이지 HTML,CSS 디자인 및 수정, 다중이미지 처리, 중고거래,동네업체(CRUD),카테고리 별 분류, 통합 및 형상관리, PPT제작

문웅기- 로그인,회원가입,아이디/비밀번호 찾기,채팅, 이벤트페이지(CRUD), 통합 및 형상관리, 발표

# 개발환경
* `Java11`
* **IDE**: STS
* **DataBase**: MySQL
* ORM: Mybatis

# 주요기능
**다중이미지 처리** - 사용자가 선택한 사진 여러장 업로드 가능
